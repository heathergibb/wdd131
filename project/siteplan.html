Site Plan for WDD131 Final Project

Site Name:

Alberta Rocky Mountains Hikes


I chose this name because the main purpose of the page is to view some info about hikes in that area.

Site Purpose:

The site provides info about a few hikes in each of the National Parks in Alberta's Rocky Mountains. It will also include some basic info about the town sites of each park; Jasper, Banff, and Waterton.

Scenarios:

  • What are some good hikes in Jasper, Banff, and Waterton?
  • Find a hike based on different filter options.
  • What are some things to do in the nearest townsite?
  • Submit a suggestion for a hike to add to the collection.
