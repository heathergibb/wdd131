Site Plan for Final WDD131 Project

Site Name:

Alberta Rocky Mountains Hikes

Site Purpose:

The site provides info about a few hikes in each of the National Parks in Alberta's Rocky Mountains. It will also include some basic info about the town sites of each park; Jasper, Banff, and Waterton.

Scenarios:

  • Which hikes are easy, moderate, or hard?
  • What are some things to do in the nearest townsite?
  • List the order of hikes from longest to shortest distance.
  • What type of hikes are they; loop, out-and-back, point-to-point?
